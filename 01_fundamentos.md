# ¿Qué es Vibe Coding?
Vibe Coding es un enfoque de desarrollo en el que la persona describe con claridad lo que desea construir y la inteligencia artificial se encarga de generar el código correspondiente. El rol del desarrollador se centra más en definir objetivos, supervisar resultados y ajustar iteraciones, en lugar de escribir manualmente cada línea de programación.
# Diferencia entre entorno web y entorno local
El entorno local es el espacio de trabajo en la propia computadora del desarrollador, donde se construye y prueba la aplicación antes de publicarla. El entorno web, en cambio, es cuando la aplicación ya está desplegada en internet y puede ser utilizada por cualquier usuario a través de un navegador.
# ¿Qué es modo Planning?
El modo Planning consiste en solicitar primero a la inteligencia artificial que elabore un plan estructurado del proyecto —incluyendo pasos, arquitectura y organización— antes de generar el código. Este enfoque es recomendable para proyectos más complejos que requieren coherencia y planificación previa.
# ¿Qué es modo Fast?
El modo Fast prioriza la rapidez y genera directamente el código a partir de la instrucción dada, sin una fase de planificación detallada. Es útil para prototipos o desarrollos simples, o instrucciones concretas ya que no se va a usar tanto razonamiento en ellas y obtendremos lo que queremos en menos tiempo.
# ¿Qué modelo usarías para generar una app completa y por qué?
Para desarrollar una aplicación completa, se recomienda utilizar un modelo avanzado con alta capacidad de contexto y razonamiento como Claude Opus 4.6 porque es de los más potentes de la familia de modelos.
